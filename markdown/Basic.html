    markdown 基础

    # markdown 基础

    # 1、标题

    # 标题名字 （‘#’的个数代表标题的级数，最多⽀持六级标题）

    # 2、字体

    # 删除线、下划线

    ~~删除的文字～～ （文字两边用‘～～’包裹起来，使用删除字体）
    效果： 删除的文字
    <u>下划线</u> (没有原生语法，使用行内html实现)
    效果：下划线

    # 斜体、加粗、斜体+加粗、高亮

    *斜体*
    效果：斜体
    **加粗**
    效果：加粗
    ***斜体+加粗***
    效果：斜体+加粗
    ==高亮==

    # 引用

    > 引用
    >> 嵌套引用
    效果:

    引用

    嵌套引用

    # 3、代码

    # 单行代码

    `console.log('hello world')`
    效果： console.log('hello world')

    # 代码块

    用 ``` 包裹一段代码，并指定一种语言（也可以不指定)

    ```javascript
    let a = 1
    let b = 2
    console.log(a+b)
    ```

    显示效果：

    let a = 1
let b = 2
console.log(a+b)

    # 4、列表

    # 无序列表

    使用星号(*)、加号(+)或是减号(-)作为列表标记，这些标记后面要添加一个空格，然后再填写内容

    - 第一项
    - 第二项
    - 第三项

    显示效果：

    • 第一项
    • 第二项
    • 第三项

    # 有序列表

    使用数字并加上 . 号来表示：

    # 列表嵌套

    只需在子列表中的选项前面添加四个空格即可

    # 5、链接

    # 外链

    [链接名称](链接地址)
    或者
    [链接名称]<链接地址>

    实例：

    [百度](https://www.baidu.com/)

    效果：

    百度 (opens new window)
